# Human Resource Analytics Dashboard

## Project Overview
The Human Resource Analytics Dashboard is an interactive Power BI project designed to analyze employee data and provide valuable HR insights. It helps organizations monitor workforce performance, employee demographics, attrition trends, and other key HR metrics to support better decision-making.

## Objectives
- Analyze employee demographics.
- Monitor employee attrition.
- Track department-wise employee distribution.
- Evaluate job satisfaction and performance.
- Support data-driven HR decisions.

## Features
- Employee Count Overview
- Attrition Analysis
- Department-wise Employee Distribution
- Gender Distribution
- Age Group Analysis
- Job Role Analysis
- Education Level Analysis
- Interactive Filters and Slicers

## Tools Used
- Microsoft Power BI
- Microsoft Excel (Dataset)

## Dataset
The dataset includes the following fields:
- Employee ID
- Age
- Gender
- Department
- Job Role
- Education
- Salary
- Attrition
- Job Satisfaction
- Years at Company
- Performance Rating

## Dashboard Visuals
- KPI Cards
- Bar Chart
- Pie Chart
- Column Chart
- Line Chart
- Slicers
- Interactive Filters

## Key Insights
- Displays total employee count and attrition rate.
- Identifies departments with high employee turnover.
- Analyzes employee distribution by age and gender.
- Compares job roles and education levels.
- Supports workforce planning and HR decision-making.

## Conclusion
The Human Resource Analytics Dashboard provides a comprehensive overview of workforce data through interactive visualizations. It helps HR teams understand employee trends, reduce attrition, improve workforce planning, and make informed business decisions.
